
# Personal Loan Propensity — Streamlit App (Final Fixed Version)
A Streamlit dashboard for Universal Bank to visualize, model, and predict customer personal loan interest.

## Quick Start
1. Upload this repo to GitHub (flat structure).
2. Go to Streamlit Cloud → Create new app → Main file: app.py.
3. Deploy!

### Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
